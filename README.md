## Development

### 1. Clone the repository

```bash
git clone https://github.com/nasken-ai/nasken.ai.git
cd nasken.ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`.

### 4. Build for production

```bash
npm run build
```
